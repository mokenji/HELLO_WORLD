como clonar?

SSH. - ir no code do repositório - copiar o link do SSH - abrir o git bash here - colocar o código git clone - após isso colocar y para aceitar e enter.

criar chave ssh

como gerar a chave?

Repositório do github, ir nas configurações - SSH and GPG keys. - abrir o terminal do git e colocar os seguintes comandos: ssh- keygen -t rsa -b 4096 -c "seu email" - cat.ssh/id_rsa.pub.
